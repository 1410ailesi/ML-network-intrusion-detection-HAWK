## 🔍 How It Works

The software analyzes network data using multiple methods:

- **One-Hot Encoding:** It turns network features into a format the AI can understand.
- **Pearson Correlation:** Finds patterns related to attacks.
- **Deep Learning Models:** Uses trained neural networks to classify threats.
- **Instance-Based Learning:** Compares new data with known cases for detection.
- **Probabilistic Models:** Assesses the likelihood of attacks to reduce false alarms.

These techniques help provide a layered approach to spotting intrusions.
